Basic Requirements :

  • Webserver running atleast PHP 5.4 and MySQL Server 5.5 / Postgres SQL Server
  • Supports PHP version PHP 5.4 to PHP 8.0.8

Note : If you don't want to / don't know how to setup the servers yourself, you can use ready made stacks like Bitnami LAMP (Linux) / WAMP (Windows) / MAMP (Mac) stacks.
